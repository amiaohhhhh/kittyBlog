# 6 switch分支语句

也是多分支语句 也可以实现多选1

语法结构 switch转换、开关、case小例子或者选项的意思

switch(表达式){
	case value1:
		执行语句1;
		break;
	case value2:
		执行语句2
		break;
	...
	default:
		执行最后的语句;
}

# 注意事项

表达式经常写成变量

变量于case里面的值相匹配的时候是全等 ===(值和数据类型一致)

没有break则不会退出 则继续执行下一个case

# switch和if else if的区别

switch通常处理case为比较固定的值  if else if处理有范围的值 灵活

switch 语句进行条件判断后直接执行到程序的条件语句，效率更高，而if-else语句有几种条件，就得判断多几次

分支比较少，if else的执行效率比较高

分支比较多，switch语句的执行效率比较高，而且结构更清晰

用来计数，跟数字有关的，三者基本相同，用for

while和do...while可以做更复杂的判断条件，比for循环灵活一些

while和do...while执行顺序不一样，while先判断后执行，do...while先执行一次，再判断执行

while和do...while执行次数不一样，do...while至少执行一次循环体,while可能一次也不执行

  • return 后面不会被执行

# break,continue,return的区别

  • break:结束当前循环体 for while
  • continue:跳出本次循环，继续执行下次循环 for while
  • return：不仅可以退出循环，还能返回return语句中的值，同时还可以结束当前的函数体内的代码

# 10 作用域

# 10.1 js作用域

代码名字在某个范围内作用和效果

目的：为了提高程序的可靠性更重要的是减少命名冲突

全局作用域：整个script标签或者一个单独的js文件

局部作用域：在函数内部就是局部作用域 在函数内部效果和作用

# 10.1.1 没有块级作用域

在es6中新增块级作用域

# 10.2 变量作用域

全局变量：在全局作用域下的变量 在函数内部没有声明直接赋值的变量

当浏览器关闭时才会销毁，比较占内存资源

局部变量：在局部作用域下的变量  程序执行完毕会销毁

# 11 预解析

  • js引擎运行js分为2步: 预解析 代码执行
    预解析 js引擎会把js里面所有的var 还有 function提升到当前作用域的最前面
    代码执行 按照代码书写的顺序从上往下执行
  • 预解析分为 变量预解析(变量提升) 和 函数预解析(函数提升)
    变量提升 把所有变量声明提升到当前的作用域的最前面 不提升赋值操作
    函数提升 把所有的函数声明提升到当前作用域的最前面 不调用函数

# 14 简单类型与复杂类型

简单类型又叫基本数据类型或者值类型，复杂类型又叫做引用类型

  • 值类型：简单数据类型/基本数据类型，在存储时变量中存储的是值本身
    string,number,boolean,undefined,null
    null返回的是一个空对象 object 如果有个变量带算存储为对象，暂时没想好则可以修改为null
  • 引用类型：复杂数据类型，在存储时变量中存储的仅仅是地址(引用)，也叫引用数据类型
    通过new关键字创建的对象(系统对象，自定义对象),object,array,date等

# 14.1 堆和栈

  • 栈(操作系统)：由操作系统自动分配释放存放函数的参数值、局部变量的值等。其操作方式类似于数据结构中的栈
    简单数据类型存放在栈里面
  • 堆(操作系统)：存储复杂类型（对象），一般由程序员分配释放，若程序员不释放，由垃圾回收机制回收
    复杂数据类型存放到堆里面
  • js没有堆和栈

# 14.2 内存分配

简单数据类型  存放在栈里面直接开辟一个空间存放值

复杂数据类型 首先在栈里面存放地址 16进制表示 地址指向堆里的数据

# 14.3 简单数据类型传参

形参可以看成是一个变量，当外面把一个值类型变量作为参数传给函数的形参时，其实是把变量在栈空间的值复制了一份给形参，那么在方法内部对形参做任何修改，都不影响到外部变量。

function fn(a){
	a++;
	console.log(a);
}
var x = 10;
fn(a);
console.log(x);
