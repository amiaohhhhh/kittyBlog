# 4.CSS的背景

# 3.1 背景图片的位置

# 3.1.1 参数是方位名词

利用background-position 属性可以改变图片在背景中的位置

background-position:x y
1

# 注意
1.方位名词不需要顺序   center right = right center

2.如果只指定一个方位名词，则第二个默认是垂直居中的

# 3.1.2 参数是精确定位

background-position:x y
x y里面放的是精确的像素值
1
2

# 注意
1.如果参数值是精确坐标，那么第一个肯定是x坐标，第二个一定是y坐标

2.如果只指定一个数值，那数值一定是x坐标，另一个默认垂直居中

# 3.1.3 参数是混合单位

第一个值是x,第二个值是y

//水平居中 距离顶部20px
background-position:center 20px;

# 4.CSS的三大特性

# 4.1 层叠性

相同选择器个设置相同的样式，此时一个样式就会覆盖另一个冲突的样式，层叠性主要解决样式冲突的问题

# 原则
1.样式冲突，遵循的原则是就近原则，哪个样式离结构近，就执行哪个样式

2.样式不冲突，不会层叠

# 4.2 继承性

子标签会继承父标签的某些样式(text-,font-,line-,以及color属性)

# 4.2.1 行高的继承

body{
	font:12px/1.5 heiti
}
//如果子元素没有行高，则继承当前子元素的行高是12*1.5倍

ID选择器 0,1,0,0

从左到右依次比较，如果数值相同，则比较下一位

# 9 浮动

# 9.1 标准流(普通流/文档流)

按标签规定默认方式排列

1、块级元素独占一行，从上到下顺序排序

div hr p h1~h6 ul ol dl form table

2、行内元素按照顺序，从左到右顺序排序，碰到父类边缘则自动换行

span a i em

# 9.2 浮动特性

1、脱离标准普通的控制浮移动到指定位置，俗称脱标

2、浮动的盒子不再保留原先的位置

3、浮动元素会具有行内块元素特性，如果块级盒子没有设置宽度，默认宽度和父级一样宽，但是添加浮动后，它的大小根据内容来决定

# 9.3 浮动元素经常和标准流父类搭配使用

1、先用标准流父元素排列上下位置，之后内部子元素采取浮动排列左右位置。

2、先设置盒子大小，之后设置盒子位置

# 9.4 一个元素浮动了，理论上其余的兄弟元素也要浮动

1、一个盒子里面有多个盒子，如果其中一个盒子浮动了，那么其他兄弟也应该浮动以防引起问题

2、浮动的盒子只会影响浮动盒子后面的标准流，不会影响前面的标准流

# 10.定位

# 10.1 静态定位

元素的默认定位方式，无定位的意思

选择器{ position:static;}
1

按照标准流特性摆放位置，没有边偏移

# 10.2 相对定位

以自身为中心,相对于它原来的位置来说

position:relative
1

特点：

相对于自己原来的位置来移动(移动位置的时候参照点是自己原来的位置)

原来在标准流的位置继续占有，后面的盒子仍然以标准流的方式对待。(不脱标，继续保留原来位置)

# 10.3 绝对定位

元素移动位置的时候，是相对于它祖先元素来说

position:absolute
1

特点：

没有祖先元素或者祖先元素没有定位，则以浏览器为定位

如果祖先元素有定位(相对、绝对、固定定位)，则以最近一级的有定位祖先元素为参考点移动位置

绝对定位不再占有标准流位置(脱标)

# 10.3.1 盒子水平居中

left:50%
margin-left=盒子的宽度一半(负)
margin:0 atuo不起效果
1
2
3

# 10.4 固定定位

元素固定于浏览器可视区的位置，可以在浏览器页面滚动元素的位置不会改变

position:fixed
1

特点：

以浏览器的可视窗口为参照点移动元素--与父元素没有关系，不随滚动条滚动

# 10.5 粘性定位

相对定位和固定定位的混合，sticky

position:sticky;
1

特点：

以浏览器的可视窗口为参照点移动元素(固定定位特点)

粘性定位占有原先位置(相对定位特点)

必须添加top left right bottom 其中一个才有效

兼容性不强，IE不支持

# 10.6 定位叠放次序 z-index

z-index:1;
1

数值可以是正整数，负整数或0，默认是auto,数值越大，盒子越靠上

如果属性值相同，则按照书写顺序，后来居上

数字后面不能加单位

只有定位的盒子才有z-index属性

# 10.7 扩展

行元素设置固定，绝对位置可以直接设置宽高属性

块级元素添加固定，绝对位置，如果不给宽度或高度，默认大小是内容的大小

脱离标准流不会触发外边距合并

浮动元素不会压住下面标准流的文字，但是绝对跟固定会

浮动主要就是做文字环绕效果，不是用来布局的本身

# 11.元素的显示与隐藏

# 11.1 display
