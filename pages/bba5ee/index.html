# 3、表单标签

使用场景：登录注册页面 收集用户信息
组成：表单域、表单控件(元素)、提示信息

# 3.1 表单域

一个包含表单元素的区域---会把它的范围内的表单元素提交给服务器

<form action="demo.php" method="POST" name="name1">
	//将该表单交给“demo.PHP”处理，以POST的形式提交，给表单域起名字为name1
</form>

# 3.2 表单元素

允许用户在表单中输入表单元素

# 3.2.1 表单元素--input

type属性:可以让input表单元素设置不同的形态

属性值 描述
button 可点击按钮
checkbox 定义复选框
file 定义输入字段和"浏览"按钮,供文件上传
hidden 定义隐藏的输入字段
image 定义图像形式的提交按钮
password 定义密码字段,该字段中的字符被掩码
radio 定义单选按钮
reset 定义重置按钮,会清除表单中的所有数据
submit 定义提交按钮,把表单数据发送到服务器
text 定义单行的输入字段,可在其中输入文本,默认宽度为20个字符
<form>
    用户名:<input type="text" name="username"> <br>
    密码:<input type="password"> 
    //单选按钮 实现多选一
    //单选按钮必须有相同的名字name,才可以实现多选一 多选多也是同一个道理
    性别:男<input type="radio"> 女 <input type="radio">
    //复选框实现多选多
    爱好: 吃饭 <input type="checkbox"> 睡觉 <input type="checkbox"> 啊豆豆<input type="checkbox">
</form>

属性 属性值 描述

maxlength 正整数 规定输入字段中的字符的最大长度

# 3.2.1.1 label标签

用于绑定一个表单元素,浏览器会自动将焦点(光标)转到或者选择对应的元素

<label for="text">用户名:</label>
<input type="text" id="text">
//for跟id中的值要一样,建立链接
<input type="radio" id="nan" name="sex"><label for="nan">男</label>
