    求最长公共前缀

    # 求最长公共前缀

    # 题目

    编写一个函数来查找字符串数组中的最长公共前缀。
    如果不存在公共前缀，返回空字符串 ""

    # 示例 1：

    输入：strs = ["flower","flow","flight"]
    输出："fl"

    # 示例 2：

    输入：strs = ["dog","racecar","car"]
    输出：""
    解释：输入不存在公共前缀。

    # 方法

    定义一个前缀（初始化）
    判断下面的数组是否跟前缀相同 若不相同把前缀中的内容往前删掉一个
    直到删完

    substring() ：方法用于提取字符串中介于两个指定下标之间的字符。
    语法：substring(from,to) 返回开始的字符但是不返回尾巴的字符

    /**
 * @param {string[]} strs
 * @return {string}
 */
var longestCommonPrefix = function(strs) {
    //极端情况下判断
    if(!strs){
        return ''
    } 
    let first_word = strs[0];//定义一个前缀
    for(let i = 0; i < strs.length;i++){
        while(strs[i].indexOf(first_word) != 0){
            first_word = first_word.substring(0, first_word.length - 1);
        }
    }
    return first_word
};
