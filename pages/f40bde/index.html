    echats统计图宽长度没有获取到

    # echats统计图宽长度没有获取到

    # 需求

    点击查询后发现由echats的统计图挤到一起了
    查看静态html页面发现是因为解析后没有计算出宽长
    导致统计图默认给了80px

    # 代码实现

    强制给个宽高+调用echats.init()作用是：用户定义的div下自动生成一个div，生成charts后，会生成一个canvas标签（创建实例）

    <div>
	<div ref='chatshow' class='mt-20 chatshow' :style="{width: chatshow_width, height: chatshow_height}"></div>
</div>

//
export default {
  data(){
    return{
      chatshow_width: (document.documentElement.clientWidth-500)+'px',
      chatshow_height: (document.documentElement.clientHeight-300)+'px',
    }
  },
  methods:{
    getList(){
      //处理数据
      if (this.myChart == null) {
        this.myChart = echarts.init(this.$refs['chatshow']);
      }
    },
  },
}

//样式
.chatshow {
  padding: 20px;
  box-sizing: border-box;
}
