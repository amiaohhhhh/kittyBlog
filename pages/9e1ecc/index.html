    表格错位的几个解决方法

    # elementui 表格错位的几个解决方法

    # 场景复现

    表头跟下面的列 错位了

    百度的时候才发现遇到这个问题的人不少 element给的解决方案是doLayout()

    # 方法一：doLayout()

    **Element 2.0.5 **已发布，开放 Table 的 doLayout 方法，用于重新计算 Table 的布局，大于这个版本的，都可以使用doLayout方法。 :::success 这个方法更适用于数据更新后/切换表格后所出现的错位 ::: 更多场景可以看这个 Element-ui中el-table出现的表格错位如何解决 - 开发技术 - 亿速云 (opens new window)

    // 在数据更新后执行
this.$nextTick(() => {
  // myTable是表格的ref属性值
  if (this.$refs.myTable && this.$refs.myTable.doLayout) {
    this.$refs.myTable.doLayout();
  }
})
